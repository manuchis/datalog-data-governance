This is the documentation for the data consent and sharing specifications that is used by [DATALOG](datalog.es) and could be used by any Data Intermediary. This platform, considered as a data sharing consent form in the absence of an official version _as per_ the Data Governance Act (EU Regulation 2022/868).

We follow the https://schema.org/URL structure. And the [MyData dictionary](https://dictionary.mydata.org).

The specifications for the documents are based on the following standards and documentation:
- https://w3c.github.io/dpv/2.1/standards/p7012/
- https://w3c.github.io/dpv/2.2/dpv/
- https://ownyourdata.github.io/soya/

For the data standards:
- https://sede.cnmc.gob.es/documentacion/home
- https://smartdatamodels.org//
